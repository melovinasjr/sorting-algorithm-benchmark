# Sorting Algorithm Benchmark

Coursework project for Data Structures Lab 13.

This project benchmarks sorting algorithms across Python, C, and Java, then records comparison results and charts for analysis.

## Coursework Note

Built as an academic project to compare sorting algorithm behavior across languages and analyze runtime differences with repeatable benchmark data.

## Features

- Python sorting benchmark implementation
- C sorting benchmark implementation
- Java sorting benchmark implementation
- Windows script to run all implementations
- CSV result files for comparison
- Charts used for report analysis

## Tech Stack

- Python
- C
- Java
- Sorting algorithms
- Benchmarking

## Run On Windows

```bat
run_all_windows.bat
```

Timing results depend on hardware, compiler/JDK/Python versions, and background processes. The included CSV files provide reference/sample benchmark data.

## Included Materials

- Source code in `python/`, `c/`, and `java/`
- Results in `results/`
- Charts in `charts/`
- Original report as `report.pdf`
