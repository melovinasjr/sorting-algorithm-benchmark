Sorting Algorithm Benchmark Submission Package

Contents:
1. python/sorting_benchmark.py      - Python implementation and benchmark
2. c/sorting_benchmark.c            - C implementation and benchmark
3. java/SortingBenchmark.java       - Java implementation and benchmark
4. results/                         - sample/reference CSV result files
5. charts/                          - comparison charts used in the report
6. Sorting_Algorithms_Report.docx   - report documentation
7. run_all_windows.bat              - Windows script to compile/run everything

How to run on Windows:
1. Install Python 3, Java JDK, and GCC/MinGW.
2. Open Command Prompt in this folder.
3. Run: run_all_windows.bat
4. Check the results folder for new CSV files.

Important note:
Timing results depend strongly on the computer, compiler/JDK/Python version, and background processes. The included CSV and report contain reference/sample benchmark data and a complete methodology. For final submission, rerun the scripts on your own computer if your professor requires machine-specific results.
